QClearTextView
==============

自定义带删除按钮的EditText,没找到好看的图片，使用的是Android自带的删除图片，有点丑<br>

准备工作
=======
Project build.gradle
--------------------
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
  Module build.gradle
  -------------------
  ```
  	dependencies {
	        implementation 'com.github.qylfzy:QClearTextView:V1.0'
	}
```
